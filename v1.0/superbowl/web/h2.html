Allgemeine Informationen zur H2 Datenbank

Note: Die H2 Datenbank liegt im Verzeichnis /opt/h2.
Im Unterverzeichnis server/bin liegt der eigentliche Datenbank-Server mit der H2 Treiberdatei h2-1.4.193.jar.
Im Unterverzeichnis store/superbowl sind die binären H2 Datenbankdateien abgelegt.
Im Unterverzeichnis backup sind die H2 Sicherungsdateien als zip-Datei(en) abgelegt.
Im Unterverzeichnis scripts liegen die SQL Scripte zur Initialisierung der H2 Datenbank.
