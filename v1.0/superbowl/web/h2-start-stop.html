Datenbank Starten/Stoppen

Die Tabelle listet alle Aktionen zum Starten/Stoppen des H2 Datenbank Servers. Zur Ausführung der Kommandos muss ein Terminalfenster geöffnet werden.
Im Terminalfenster musst Du Dich im Home-Verzeichnis des angemeldeten Benutzers befinden!

Aktion Kommando (Terminal) Message
H2 Datenbank Starten ./h2-daemon.sh start h2 database server started. Running on Port 9092 - Success
H2 Datenbank Stoppen ./h2-daemon.sh stop SIGTERM send to process {processnumber}
h2 database server stopped - Success
H2 Datenbank Status ./h2-daemon.sh status h2 database server [not] running
