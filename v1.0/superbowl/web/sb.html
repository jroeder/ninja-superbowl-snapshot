Allgemeine Informationen zur Superbowl Applikation

Note: Die Superbowl Applikation liegt im Verzeichnis /opt/superbowl.
Im Unterverzeichnis bin liegen die Superbowl Archiv-Dateien superbowl.jar und superbowl-javadoc.jar sowie die Superbowl Log-Datei superbowl.log.
Im Unterverzeichnis web sind alle HTML-Dateien für die Online Superbowl Help Applikation abgelegt.
Im Unterverzeichnis img befinden sich die Verzeichnisse bowl, gallery und user.

!!! Ich empfehle Dir, alle von Dir z.B. mit der Applikation Gimp erzeugten Bilder von Schalen in das Verzeichnis /opt/superbowl/img/bowl abzulegen !!!
Von dort fügst Du neue Bilder dann, wie unter Administration - Bildverwaltung beschrieben, in die Superbowl Archivdatei ein.

Beim Starten der Superbowl Applikation mit sudo wird noch das Passwort des aktuell am Linux System angemeldeten Benutzers abgefragt. In Deinem Fall musst Du das Passwort des Benutzers thomas eingeben!

Im Browser Deiner Wahl (Empfehlung: Opera) die URL http://localhost:9000/superbowl eingeben. Voila - Superbowl hero starting!!!

Superbowl Starten/Stoppen

Die Tabelle listet alle Aktionen zum Arbeiten mit der Superbowl Applikation. Zur Ausführung der Kommandos muss ein Terminalfenster geöffnet werden.
Im Terminalfenster musst Du Dich im Home-Verzeichnis des angemeldeten Benutzers befinden!

Aktion Kommando (Terminal) Message
Superbowl Starten sudo ./superbowl.sh start Starting Ninja Applikation: Superbowl - Success
Superbowl ReStarten sudo ./superbowl.sh restart Stopping Ninja Applikation: Superbowl - Success
Starting Ninja Applikation: Superbowl - Success
Superbowl Stoppen sudo ./superbowl.sh stop Stopping Ninja Applikation: Superbowl - Success
Superbowl Status abfragen sudo ./superbowl.sh status Ninja application Superbowl [not] running
